eRSD Version 3 Specification Bundle

eRSD v3 includes the queries necessary for producing both CDA eICR R1.1 and R3.1.1 documents and is the recommended and actively supported specification for all implementers.

eRSD VERSION 3 RELEASE DESCRIPTION
Important Notice: RCTC Spreadsheet Deprecation

The RCTC Spreadsheet will be deprecated and no longer available for implementing updates by late 2026. If your implementation currently relies on the spreadsheet format, we strongly recommend beginning the transition to a FHIR-based implementation.
