eRSD VERSION 3 RELEASE DESCRIPTION

eRSD Version (v3) includes the queries necessary for producing both CDA eICR R1.1 and R3.1.1 documents and is the recommended and actively supported specification for all implementers.

eRSD VERSION 2 RELEASE DESCRIPTION

eRSD Version (v2) introduced the Specification Library and manifest-based structure used by eRSD v3. It is supported for existing implementations but will no longer be maintained after January 31, 2026, including no new updates or RCTC content.

eRSD VERSION 1 RELEASE DESCRIPTION

eRSD v1 supported the earliest eCR 1.x implementations and used date-based versioning. It is supported for existing implementations but will no longer be maintained after January 31, 2026, including no new updates or RCTC content.
